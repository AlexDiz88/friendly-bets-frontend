1. Статус сезона. При смене статуса на ACTIVE, остальные ACTIVE ставить на паузу. (средний)
2. Регистрация на турнир. После регистрации убрать кнопку, вместо неё поставить сообщение о том что пользователь уже зарегистрированы (низкий)
3. Сделать список пользователей, уже зарегистрированных на турнир на странице регистрации (низкий)
4. Редактирование лиг (низкий). Значок рядом с названием
5. Редактирование команд (средний)
6. Добавление логотипа команды через интерфейс и загрузку с ПК (низкий)
7. Обновление отображения списка команд лиги после добавления / удаления (средний)
8. Удаление команды из лиги на "крестик" (в случае ошибочного добавления) с подтверждением. Плюс проверка на состояние турнира (можно удалять только в CREATED и SCHEDULED) (низкий)
10. Отображение текста ошибки если при добавлении команды не выбран сезон, лига или команда (низкий)
11. Смена email - проверка подлинности пользователя? подтверждение email по ссылке
12. Смена пароля
13. В различных списках (лиг, команд итд.) добавить текст "Нет команд", если отсутствуют (очень низкий)
14. Проверка валидности email при регистрации (наличие @ итд)
15. При подтверждении ставки отобразить все данные. Для этого возвращать через коллбэк полные объекты а не ID
16. На странице добавления ставок сделать переключатель для типа "Тура" для матчей плей-офф
17. Для матчей плей-офф пересмотреть в файле BetsCheck проверку валидности счёта
